## Setup
Here are the steps to setup Xcode project using [CocoaPods](http://cocoapods.org):

1. Install CocoaPods:
    ```bash
    gem install cocoapods
    ```

2. Setup Cocoapods:
    ```bash
    pod setup
    ```

3. Install SparkSDK and other dependencies from your project directory:

    ```bash
    pod install
    ```
